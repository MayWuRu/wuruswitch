# WuRuSwitch

โปรแกรมสลับภาษาและตัวช่วยพิมพ์สำหรับแก้ปัญหาการพิมพ์ผิดภาษา (Thai <-> English) 
ออกแบบมาเพื่อให้การทำงานมีความต่อเนื่อง ไม่สะดุด และมีตัวช่วยแจ้งเตือนภาษาที่กำลังพิมพ์ โดยไม่รบกวนการทำงานของโปรแกรมอื่นๆ (เช่น เครื่องสแกนบาร์โค้ด POS)

## ฟีเจอร์หลัก
- **แปลงข้อความที่พิมพ์ผิดภาษา (Auto-Translate):** เปลี่ยนภาษาของข้อความที่คลุมดำไว้ได้อย่างรวดเร็วและแม่นยำ (เช่น พิมพ์ `l;ylfu` สลับเป็น `สวัสดี`)
- **สลับภาษาแป้นพิมพ์อัตโนมัติ (Auto-Switch Layout):** หลังจากแปลงข้อความเสร็จ โปรแกรมจะสลับภาษาของแป้นพิมพ์ให้ตรงกับข้อความที่แปลง เพื่อให้คุณสามารถพิมพ์ต่อได้ทันทีโดยไม่ต้องกดสลับภาษาเอง
- **ธงแจ้งเตือนภาษาที่เคอร์เซอร์ (Cursor Language Indicator):** แสดงธงชาติ (อเมริกา 🇺🇸 / ไทย 🇹🇭) ขนาดเล็กห้อยที่เคอร์เซอร์ขณะพิมพ์ เพื่อให้ทราบทันทีว่ากำลังใช้ภาษาอะไรอยู่ และมีสัญลักษณ์แจ้งเตือน "A" สีแดงหากเผลอเปิด CapsLock ไว้
- **บังคับปิด CapsLock เสมอ (Always Disable CapsLock):** ฟีเจอร์สำหรับคนที่มักจะเผลอไปโดนปุ่ม CapsLock ระบบจะทำการดึงสถานะกลับมาปิดให้ทันทีโดยอัตโนมัติ
- **บังคับเปิด NumLock เสมอ (Always Enable NumLock):** ป้องกันการเผลอไปโดนปุ่ม NumLock จนพิมพ์ตัวเลขไม่ได้ ระบบจะล็อกให้เปิดเสมอ
- **ปรับแต่งได้ตามใจ (Settings):** สามารถตั้งค่าปุ่มลัด (Hotkey) ได้ด้วยตนเอง (รองรับ Ctrl, Alt, Shift, Win) และเปิด/ปิดฟีเจอร์ต่างๆ ด้านบนได้อิสระ
- **เบาและปลอดภัย:** ทำงานอยู่เบื้องหลังใน System Tray ไม่เกะกะหน้าจอ สร้างจาก C# .NET ไม่มีการใช้ Keyboard Hook ในส่วนหลัก ทำให้ระบบคีย์บอร์ดเสถียร ไม่รวนกับ On-Screen Keyboard (OSK)

## วิธีใช้งานเบื้องต้น
โปรแกรมออกแบบมาให้ใช้งานได้ 2 รูปแบบตามความสะดวก:

**แบบที่ 1: แก้ไขคำล่าสุดที่พิมพ์ผิด (รวดเร็วที่สุด)**
1. หากพิมพ์ผิดแล้วเพิ่งรู้ตัว ไม่ต้องคลุมดำใดๆ
2. กดปุ่ม `F6` (หรือปุ่มลัดที่ตั้งไว้) ได้เลย
3. โปรแกรมจะลบคำที่พิมพ์ผิดล่าสุด แปลงภาษา และพิมพ์คำที่ถูกต้องให้ พร้อมสลับภาษาแป้นพิมพ์ให้คุณพิมพ์ต่อทันที

**แบบที่ 2: แก้ไขข้อความยาวๆ หรือทั้งประโยค**
1. คลุมดำข้อความยาวๆ ที่พิมพ์ผิด
2. กดปุ่ม `F6` (หรือปุ่มลัดที่ตั้งไว้)
3. โปรแกรมจะคัดลอก แปลงภาษา วางข้อความที่ถูกต้องทับลงไป และสลับภาษาแป้นพิมพ์ให้ทันที

## วิธีการปรับแก้และติดตั้ง
สำหรับนักพัฒนา:
- สามารถรันคำสั่ง `C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe /target:winexe /win32icon:logo.ico /out:WuRuSwitch.exe Program.cs` เพื่อ Compile โปรแกรมได้โดยตรงผ่าน `csc.exe` ที่ติดมากับ Windows
