# WuRuSwitch

โปรแกรมสลับภาษาสำหรับแก้ปัญหาการพิมพ์ผิดภาษา (Thai <-> English) 
ออกแบบมาโดยไม่ต้องใช้ Keyboard Hook ทำให้ปลอดภัยและไม่รบกวนการทำงานของโปรแกรมอื่นๆ (เช่น เครื่องสแกนบาร์โค้ด POS)

## ฟีเจอร์หลัก
- เปลี่ยนภาษาของข้อความที่คลุมดำไว้ได้อย่างรวดเร็วและแม่นยำ
- รองรับการตั้งค่าปุ่มลัด (Hotkey) ได้ด้วยตนเอง (รองรับ Ctrl, Alt, Shift, Win)
- ทำงานอยู่เบื้องหลังใน System Tray ไม่เกะกะหน้าจอ
- ไม่ดักจับข้อมูลการพิมพ์ ปลอดภัย 100%
- สร้างจาก C# .NET ทำให้โปรแกรมมีขนาดเล็ก เบา และทำงานได้รวดเร็วทันที

## วิธีใช้งานเบื้องต้น
1. คลุมดำข้อความที่พิมพ์ผิด
2. กดปุ่ม `F6` (หรือปุ่มลัดที่ตั้งไว้)
3. โปรแกรมจะคัดลอก แปลงภาษา และวางข้อความที่ถูกต้องทับลงไปทันที

## วิธีการปรับแก้และติดตั้ง
สำหรับนักพัฒนา:
- สามารถรัน `csc.exe /target:winexe /out:WuRuSwitch.exe Program.cs` เพื่อ Compile โปรแกรมได้โดยตรงผ่าน `csc.exe` ที่ติดมากับ Windows (ในโฟลเดอร์ `C:\Windows\Microsoft.NET\Framework64\`)

*สร้างสรรค์โดยทีม WuRu*
